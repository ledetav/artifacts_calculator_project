# Genshin Impact AI Build Optimizer (Название проекта)

Проект представляет собой мобильное приложение для Android, предназначенное для помощи игрокам в управлении артефактами, оружием и персонажами из игры Genshin Impact. Ключевая особенность — использование **искусственного интеллекта (Gemini API)** для автоматического создания оптимальных билдов и командных сборок на основе имеющихся у пользователя предметов и персонажей.

## Технологический стек

* **Фреймворк:** Flutter (Язык: Dart)
* **Распознавание текста (OCR):** Google ML Kit (Text Recognition)
* **Искусственный интеллект:** Google Gemini API
* **База данных:** SQLite

## Будущие улучшения (Backlog)

### **Автоматическое обновление игровой базы данных (Wiki Parser)**

**Цель:** Создать механизм, который будет автоматически проверять наличие новых персонажей, оружия и артефактов на англоязычной вики Genshin Impact (Fandom) и добавлять их в локальную базу данных приложения. Это позволит поддерживать приложение в актуальном состоянии без необходимости выпускать обновления в Google Play.

**Подход:** Использовать официальный **Fandom API**, который предоставляет структурированные данные в формате JSON.

#### **План разработки для Wiki Parser**

#### **Спринт 1: Исследование и проектирование**

* [ ] Изучить документацию Fandom API для Genshin Impact Wiki.
* [ ] Определить конечные точки (endpoints) для получения списков персонажей, оружия и артефактов.
* [ ] Проанализировать структуру JSON-ответов от API, чтобы понять, как извлекать необходимые данные (имя, редкость, статы, описания и т.д.).
* [ ] Спроектировать логику сравнения данных: как приложение будет определять, какие сущности являются новыми и требуют добавления в локальную БД.
* [ ] Спроектировать UI для этой функции (например, кнопка "Проверить обновления" в настройках).

#### **Спринт 2: Реализация парсера для персонажей**

#### **Спринт 3: Расширение парсера и интеграция в UI**
