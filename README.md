# Genshin Impact AI Build Optimizer

Проект представляет собой нативное мобильное приложение для Android, предназначенное для помощи игрокам в управлении артефактами, оружием и персонажами из игры Genshin Impact. Ключевая особенность — использование **искусственного интеллекта (Gemini API)** для автоматического создания оптимальных билдов и командных сборок на основе имеющихся у пользователя предметов и персонажей.

## Технологический стек (Android Studio)

* **IDE:** **Android Studio**
* **Язык:** **Kotlin**
* **Архитектура:** **MVVM (Model-View-ViewModel)**
* **UI:** **Jetpack Compose**
* **Асинхронность:** **Kotlin Coroutines**
* **База данных:** **Room Persistence Library**
* **Сеть:** **Retrofit & OkHttp**
* **Распознавание текста (OCR):** **Google ML Kit (Text Recognition)**
* **Искусственный интеллект:** **Google Gemini API**
